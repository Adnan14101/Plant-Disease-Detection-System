# 🌿 Plant Disease Detection using Machine Learning

## 📌 Project Overview
This project is a **Plant Disease Detection System** that uses **Machine Learning / Deep Learning** techniques to identify diseases in plant leaves from images.

It helps farmers and users detect plant diseases early and take proper action to improve crop health.

---

## 🚀 Features
- 🌱 Detect plant diseases from leaf images  
- 🧠 Uses trained ML/DL model for classification  
- 📊 Supports multiple classes (healthy + diseased)  
- 🖼️ Image-based prediction  
- ⚡ Fast and accurate results  

---

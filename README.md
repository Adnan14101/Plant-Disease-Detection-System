# 🌿 Plant Disease Detection using Machine Learning

## 📌 Project Overview
This project is a **Plant Disease Detection System** that uses **Machine Learning / Deep Learning** techniques to identify diseases in plant leaves from images.

It helps farmers and users:
- Detect plant diseases early  
- Improve crop quality  
- Take preventive actions  

---

## 🚀 Features
- 🌱 Detects diseases from leaf images  
- 🧠 Uses trained CNN model for classification  
- 📊 Supports multiple classes (healthy + diseased)  
- 🖼️ Image-based prediction system  
- ⚡ Fast and efficient results  

---

## 🛠️ Technologies Used
- Python  
- TensorFlow / Keras  
- OpenCV  
- NumPy & Pandas  
- Matplotlib
